# 5. Самая длинная палиндромная подстрока
Дана строка `s`, верните самую длинную палиндромную подстроку в `s`.

## Пример 1:
<pre>
<b>Ввод:</b> s = "babad"
<b>Вывод:</b> "bab"
<b>Пояснение:</b> "aba" тоже корректный ответ.
</pre>

## Пример 2:
<pre>
<b>Input:</b> s = "cbbd"
<b>Output:</b> "bb"
</pre>

## Ограничения:
- `1 <= s.length <= 1000`
- `s` содержит только цифры и английские буквы.
